- 获取组件列表信息

**请求URL：** 
- ` ${schema}://${ip}:${port}/public/deleteBatch?publicCodes=xxx `
  
**请求方式：**
- GET 

**参数：**

| publicCodes | array | 否 | 无 | 公众号 id，数组形式|

    "time": "20180224170008"
