**简要描述：** 

- 获取组件详细信息

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/public/get?publicCode=xxx `
  
**请求方式：**
- GET

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |

**返回参数说明** 

| 参数名 | 类型 | 说明 |
| :-- | :-- | :-- |
| publicCode | string | 公众号主键 |
