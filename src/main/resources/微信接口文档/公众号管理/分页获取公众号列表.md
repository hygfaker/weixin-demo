**简要描述：** 

- 获取组件详细信息

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/public/getPage?currentPage=xxx&pageSize=xxx `
  
**请求方式：**
- GET

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| currentPage | int | 是 | 1 | 当前页数 |
| pageSize | int | 是 | 10 | 每页显示的数量 |
