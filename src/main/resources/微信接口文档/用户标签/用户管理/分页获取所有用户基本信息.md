**简要描述：** 

- 获取组件详细信息

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/user/userInfoPage?publicCode=xxx&currentPage=xxx&pageSize=xxx `
  
**请求方式：**
- GET

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |
| currentPage | int | 是 | 1 | 当前页 |
| pageSize | int | 市 | 5 | 每页显示的数量 |
