**简要描述：** 

- 取消黑名单

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/tag/get?publicCode=xxx&tagid=xxxx `
  
**请求方式：**
- GET

**url参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |
| tagid | int | 否 | 无 | 标签 id |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180225110257"
}
```
