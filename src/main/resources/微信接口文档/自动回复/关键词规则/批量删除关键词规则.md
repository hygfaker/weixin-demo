**简要描述：** 

- 批量删除关键词规则


**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/reply/deleteRuleBatch?publicCode=xxx&ruleCodes=xxx `
  
**请求方式：**
- GET 

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号主键 |
| ruleCodes | array | 否 | 无 | 关键词规则数组 |


 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180227155219"
}
```
