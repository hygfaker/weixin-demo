**简要描述：** 

- 获取关键词规则列表


**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/reply/getRulePage?publicCode=xxx&currentPage=xxx&pageSize=xxx
  
**请求方式：**
- GET 

**URL参数：** 

| 参数名 | 类型 | 可空添加关键词规则 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号主键 |
| currentPage | number | 是 | 1 | 当前页 |
| pageSize | number | 是 | 5 | 每页显示的数量 |
