**简要描述：** 

- 关闭会话

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/kefu/sessionClose`
  
**请求方式：**
- POST

**参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |
| openid | string | 否 | 无 | 用户 openid |
| kfAccount | string | 否 | 无 | 完整客服帐号，格式为：帐号前缀@公众号微信号 |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180225101805"
}
```
