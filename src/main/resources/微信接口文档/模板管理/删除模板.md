**简要描述：** 

- 获取模板列表

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/template/delete?templateId=xxx&publicCode=xxx`
  
**请求方式：**
- GET

**url参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| templateId | string | 否 | 无 | 模板消息 id |
| publicCode | string |  否 | 无|  公众号主键 |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180225181401",
}
```
