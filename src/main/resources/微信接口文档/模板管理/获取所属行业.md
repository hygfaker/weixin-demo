**简要描述：** 

- 获取所属行业

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/template/industry?publicCode=xxx`
  
**请求方式：**
- GET

**参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string |  否 | 无 | 公众号主键 |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180225182115",
    "data": {
        "primaryIndustry": {
            "firstClass": "IT科技",
            "secondClass": "互联网|电子商务"
        },
        "secondIndustry": {
            "firstClass": "IT科技",
            "secondClass": "电子技术"
        }
    }
}
```
