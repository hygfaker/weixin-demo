**简要描述：** 

- 获取模板列表

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/template/list?currentPage=xxx&pageSize=xxx&publicCode=xxx`
  
**请求方式：**
- GET

**url参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |
| currentPage | int |  是 | 1 | 当前页数 |
| pageSize |  int | 是 | 5 | 每页显示数量 |
