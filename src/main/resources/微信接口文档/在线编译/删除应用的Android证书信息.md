- 删除应用的Android证书信息

**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/account/apps/:appId/certs/android `
  
**请求方式：**
- DELETE

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| appId | number | 否 | 无 | 应用ID |

**参数：** 

无
