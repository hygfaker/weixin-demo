**简要描述：** 

- 删除用户自定义组件

**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/account/components/:componentId `
  
**请求方式：**
- DELETE 

**参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| componentId | number | 否 | 无 | 组件ID |

 **返回示例**

``` 
{
    "status": 200,
    "desc": "执行成功",
    "timestamp": 1509677076900
}
```
