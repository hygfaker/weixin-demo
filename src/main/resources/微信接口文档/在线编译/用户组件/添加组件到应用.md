**简要描述：** 

- 添加组件到应用

**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/account/apps/:appId/components `
  
**请求方式：**
- POST 

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| appId | number | 否 | 无 | 应用ID |

**参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| componentId  | string | 否 | 无 | 组件ID，可以是公共组件ID，也可以是自定义组件ID |
| versionId  | string | 否 | 无 | 组件的版本ID |

 **返回示例**

``` 
{
    "status": 200,
    "desc": "执行成功",
    "timestamp": 1509679684861
}
```
