**简要描述：** 

- 按分组群发消息


**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/push/getPage?publicCode=xxx `
  
**请求方式：**
- GET 

**URL数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号主键 |

| data | array | 定点消息数组 |
