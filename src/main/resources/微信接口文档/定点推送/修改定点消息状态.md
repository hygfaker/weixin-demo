**简要描述：** 

- 修改定点消息状态


**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/push/updateFlag?pushCode=xxx&pushFlag=xxx&publicCode=xxx `
  
**请求方式：**
- GET 

**URL参数：** ：

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号主键 |
| pushCode | string | 否 | 无 | 定点消息主键 |
| pushFlag | number | 否 | 无 | 是否开启该推送。0为关闭，1位开启。 |


 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180226100350"
}
```
