**简要描述：** 

- 用户登录接口

**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/material/media?mediaid=xxx&publicCode=xxx `
  
**请求方式：**
- GET 

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| mediaid | String | 否 | 无 |  多媒体素材 id |
| publicCode | string | 否 | 无 | 公众号主键 |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180226155024",
}
```
