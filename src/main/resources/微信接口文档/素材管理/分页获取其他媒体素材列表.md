**简要描述：** 

- 用户登录接口

**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/material/mediaList?type=xxxpublicCode=xxx&offest=xxx&count=xxx `

**请求方式：**
- GET

**URL参数：**

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |

| type | string | 否 | 无 | 素材的类型，图片（image）、视频（video）、语音 （voice）、图文（news） |
| publicCode | string | 否 | 无 | 公众号主键 |
| offest | int | 否 | 无 | 从全部素材的该偏移位置开始返回，0表示从第一个素材 返回 |
| count | int | 否 | 无 | 返回素材的数量，取值在1到20之间 |
