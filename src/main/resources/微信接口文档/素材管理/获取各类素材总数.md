**简要描述：** 

- 用户登录接口

**请求URL：** 
- ` ${schema}://${ip}:${port}/modu/api/material/allCount?publicCode=xxx `
  
**请求方式：**
- GET 

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号主键 |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180226165926",
    "data": {
        "voiceCount": 1,
        "videoCount": 1,
        "imageCount": 2,
        "newsCount": 4
    }
}
```

 **返回参数说明** 

| 参数名 | 类型 | 说明 |
| :-- | :-- | :-- |
| voiceCount | int | 语音素材数量 |
| videoCount | int | 视频素材数量 |
| imageCount | int | 图片素材数量 |
| newsCount | int | 图文素材数量 |
