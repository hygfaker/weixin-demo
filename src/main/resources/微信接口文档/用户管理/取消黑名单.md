**简要描述：** 

- 获取组件详细信息

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/user/pushBlackList `
  
**请求方式：**
- POST

**参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |
| openids | string | 否 | 无 | 用户 openid，多个 openid 用英文逗号隔开 |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180225101805"
}
```

参数参考参数信息
