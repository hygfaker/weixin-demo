**简要描述：** 

- 获取组件详细信息

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/user/blacklist?publicCode=xxx `
  
**请求方式：**
- GET

**URL参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180225101540",
    "data": []
}
```


**返回参数说明** 

| 参数名 | 类型 | 说明 |
| :-- | :-- | :-- |
| data | array | 用户 openid 数组 |
