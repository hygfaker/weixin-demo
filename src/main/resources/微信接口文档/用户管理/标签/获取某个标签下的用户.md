- ` ${schema}://${ip}:${port}/wechat/api/tag/userinfoOfTagPage?publicCode=xxx&&tagID=xxx&pageSize=xxx&currentPage=xxx `
  
**请求方式：**
- GET

**url参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |
| tagID | int | 否 | 无 | 标签 id |
| currentPage | string | 是 | 1 | 当前页 |
| pageSize | int | 是 | 5 | 每页显示数量 |
