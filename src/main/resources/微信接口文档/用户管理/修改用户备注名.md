**简要描述：** 

- 取消黑名单

**请求URL：** 
- ` ${schema}://${ip}:${port}/wechat/api/user/updateRemark `
  
**请求方式：**
- POST

**参数：** 

| 参数名 | 类型 | 可空 | 默认值 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| publicCode | string | 否 | 无 | 公众号id |
| openid | string | 否 | 无 | 用户 openid |
| remark | string | 否 | 无 | 新的备注名 |

 **返回示例**

``` 
{
    "desc": "执行成功",
    "status": 200,
    "time": "20180225101805"
}
```
